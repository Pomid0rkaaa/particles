# Particle simulation

## Controls

| `Space`                             | Pause/Resume                |
|-------------------------------------|-----------------------------|
| `C`                                 | Clear                       |
| `1`                                 | Mode: None                  |
| `2`                                 | Mode: Push                  |
| `3`                                 | Mode: Attract               |
| `4`                                 | Mode: Orbit                 |
| `Mouse Button Left`                 | Spawn point                 |
| `Mouse Button Right`                | Delete point near cursor    |
| `Left Shift` + `Mouse Button Left`  | Spawn 5 points              |
| `Left Shift` + `Mouse Button Right` | Delete 5 points near cursor |
